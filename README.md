# PTA_AutoErase

## Intro
This is my Automatic Whiteboard Eraser project that I call it **AutoErase** and powered by Arduino. This is a Final Year Project for my collage 

## Component used
- Arduino Uno x1
- Breadboard x1
- Hobby DC motor x2
- Jumper wires
- L298N Motor Driver x1
- Ultrasonic Sensor x2
- Pushbutton x1
- Resistors
